Introducción
============
Descripción: arquetipo de un servicio de tipo Negocio, el cual contiene la configuración 
básica que permite ejemplificar como exponer un servicio vía SCA mediante 
una interfaz de tipo Java y a su vez consumir un recurso de negocio (base de datos u otro tipo). 

Estructura del Proyecto:

XML Relevantes:
---------------
 - switchyard.xml
 - route.xml
 
 Interfaces:
 -----------
 JAVA - PersonaServiceInterface.java:
  Clase dummy utilizada a modo de ejemplo por binding service y binding reference en archivo descriptor switchyard.xml. Es sólo una guía para ejemplificar estructura del proyecto.
